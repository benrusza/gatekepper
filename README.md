# GateKeeper: Your Gateway to Video Freedom
=============================================
## Overview
GateKeeper is an Android app designed to help you break free from the endless scrolling of algorithmic feeds on proprietary video platforms. With GateKeeper, you can access and watch videos from these platforms in a clean and distraction-free environment, promoting a healthier and more mindful viewing experience.

## Features
* **Decouple from algorithmic feeds**: Watch videos without being sucked into endless scrolling and recommendations.
* **Clean and minimalistic interface**: Focus on the content, not the distractions.
* **Support for multiple platforms**: Access videos from various proprietary platforms.

## Getting Started
To get started with GateKeeper, follow these steps:

1. Clone the repository: `git clone https://github.com/benrusza/GateKeeper.git`
2. Open the project in Android Studio.
3. Build and install the app on your Android device.

## How it Works
GateKeeper uses a combination of web scraping and API integration to fetch videos from proprietary platforms. The app then presents the videos in a clean and minimalistic interface, allowing you to watch and enjoy the content without distractions.

## Benefits
By using GateKeeper, you can:

* Reduce screen time and avoid mindless scrolling.
* Focus on the content you want to watch, without being recommended more of the same.
* Support a healthier and more mindful viewing experience.
